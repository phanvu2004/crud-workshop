## Chuẩn bị
+ Nodejs(v18.9+)
+ terminal
## Tải angular cli 17
chạy command sau trong terminal: 
```
npm i -g @angular/cli@17
```
Vậy là chúng ta đã tải xong angular 17. Tiếp theo, chúng ta sẽ tìm hiểu về cách khởi tạo project.
